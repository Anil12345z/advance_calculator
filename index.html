Description & help-

-:For symbols (log, sin, cos, π, √,etc)- First enter the number then click on desired symbol for answer:-

1. Symbol % means (percentage) number divide by 100.

2. x^x returns the value of x to the power x.

3. Log10 returns value of lox x with base 10.

4. Log2 returns value of lox x with base 2.

5. R-OFF returns the round off value.

6. Sin returns the value of sin with angle in degree.

7. Cos returns the value of cos with angle in degree.

8. π returns the value of PI.

9. √ returns the square-root of number.
